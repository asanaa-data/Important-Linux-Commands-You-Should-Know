### **Description:**
> ***Display system date and time.***
---------------------------------------

<strong>Command: </strong>

```linux
date
```
----------------------------------------

### **Description:**
> ***Display calendar.***
---------------------------------------

<strong>Command: </strong>

```linux
cal
```
----------------------------------------

### **Description:**
> ***Display date, time and calendar.***
---------------------------------------

<strong>Command: </strong>

```linux

```
----------------------------------------
